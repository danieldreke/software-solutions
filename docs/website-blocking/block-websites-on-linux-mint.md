1. Install *Domain Blocker*
   ```sh
   sudo apt install mintnanny
   ```
2. Open *Domain Blocker* and add websites you want to block
3. Test by opening blocked websites in your browser
    - If blocked websites still accessible, clear browser cache:
        - Brave: Open `brave://net-internals/#dns` <a href="brave://net-internals/#dns">🔗</a> and click <kbd>Clear host cache</kbd>
        - Firefox: Open `about:networking#dns` <a href="about:networking#dns">🔗</a> and click <kbd>Clear DNS Cache</kbd>
