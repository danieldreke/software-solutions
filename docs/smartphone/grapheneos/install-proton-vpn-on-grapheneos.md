# Install Proton VPN on GrapheneOS

1. Install F-Droid
    1. Open browser Vanadium
    2. Go to [f-droid.org](https://f-droid.org/)
    3. Download `F-Droid.apk` by clicking button `Download F-DROID` or [2]
    4. Install
2. Install Aurora
3. Install proton vpn
