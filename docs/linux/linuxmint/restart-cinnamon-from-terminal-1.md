1. Switch to terminal 1 via `CTRL-ALT-F1`
1. Execute `killall -9 cinnamon`
1. Execute `export DISPLAY=:0.0 && cinnamon --replace &`
1. Switch back to desktop via `CTRL-ALT-F8`
